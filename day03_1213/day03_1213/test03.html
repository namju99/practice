animate

effect method는 애니메이션이 큐메모리에 저장 mouseover, mouseout{ 이벤트 다중 발생 } mouseenter, mouseleave{ 사용 추천 } stop() : 큐에 누적된 애니메이션을 비움 
            animate({ css속성 : css값 , ........}, speed, ease, callback )
            애니메이션이 되는 css속성은  값이 숫자인 경우만 가능 
            margin, padding, width, height, letter-spacing, word-spacing,
            top, left, right, bottom, opacity, text-indent  
            ease(가속): 'swing', 'linear' 2개 속성만 있음 
            callback : 애니메이션 처리후 할일
